<section id="model-zoo-api">
<h2>Model Zoo API<a class="headerlink" href="#model-zoo-api" title="Permalink to this headline">#</a></h2>
<p>The Model Zoo API provides a robust set of reference layers for both transformers and vision models within the <a class="reference internal" href="../_autosummary/cerebras.modelzoo.layers.html#module-cerebras.modelzoo.layers" title="cerebras.modelzoo.layers"><code class="xref py py-mod docutils literal notranslate"><span class="pre">cerebras.modelzoo.layers</span></code></a> module. This module serves as a foundational resource for building and customizing your own models.</p>
<p>For task-specific requirements, the Model Zoo API offers specialized modules. The <a class="reference internal" href="../_autosummary/cerebras.modelzoo.models.nlp.html#module-cerebras.modelzoo.models.nlp" title="cerebras.modelzoo.models.nlp"><code class="xref py py-mod docutils literal notranslate"><span class="pre">cerebras.modelzoo.models.nlp</span></code></a> module includes layers and components tailored for natural language processing tasks, while the <a class="reference internal" href="../_autosummary/cerebras.modelzoo.models.vision.html#module-cerebras.modelzoo.models.vision" title="cerebras.modelzoo.models.vision"><code class="xref py py-mod docutils literal notranslate"><span class="pre">cerebras.modelzoo.models.vision</span></code></a> module caters to computer vision applications. These modules streamline the development process by offering pre-configured, high-performance layers designed to meet the unique demands of NLP and vision tasks.</p>
</section>
<section id="trainer-api">
<h2>Trainer API<a class="headerlink" href="#trainer-api" title="Permalink to this headline">#</a></h2>
<p>The Trainer API provides a simple but flexible interface to run any type of training and/or validation. The Trainer is designed to be easily extendable through the Callback mechanism and most of the Trainerâ€™s core functionality is already implemented via a set of core callbacks available to users. Once configured, a training or validation loop can be run by simply calling <code class="docutils literal notranslate"><span class="pre">trainer.fit</span></code> or <code class="docutils literal notranslate"><span class="pre">trainer.validate</span></code>.</p>
